# Исследовательский и статистический анализ данных сервиса Яндекс Афиша
[Код проекта](https://github.com/KTOTO426/Projects/blob/main/Yandex_afisha_analyze.ipynb), [Дэшборд проекта](https://datalens.yandex/q4lud5ietz72b)
## Цель проекта
### Цель проекта - используя данные сервиса Яндекс Афиша провести исследовательский анализ данных, чтобы выявить инсайты об изменении пользовательских предпочтений и популярности событий осенью 2024 года, а также проверить гипотезы о разнице в поведении пользователей с мобильными и стационарными устройствами.
## Описание данных
### Яндекс Афиша - сервис бронирования билетов. С его помощью пользователи могут узнавать информацию о мероприятиях в разных городах и покупать на них билеты. Сервис сотрудничает с партнёрами — организаторами мероприятий и билетными операторами, которые предоставляют информацию о событиях и выставляют билеты на продажу.
### Результат
### Выявлены 5 ключевых паттерна изменения поведения и подготовлены рекомендации по адаптации контента под сегменты.
## Стек
### SQL, Python (pandas, matplotlib, seaborn), DataLens

# Исследование стартапов
[Код проекта](https://github.com/KTOTO426/Projects/blob/main/Startup_analyze_ipynb_.ipynb)
## Цель проекта
### Цель проекта - Подготовить датасет и проверить, что данные в нём соответствуют, а также ответить на вопросы заказчика как о предобработке, так и о значении данных для бизнеса.
## Описание данных
### Финансовая компания, которая предоставляет льготные займы стартапам, хочет войти на инвестиционный рынок с прицелом на покупку, развитие и последующую перепродажу перспективных стартапов. Для разработки модели бизнеса нужна информация. Данные неполны, поэтому заказчик, кроме ответа на некоторые интересующие его вопросы, просит аккуратно относиться к получаемым в процессе исследования значениям показателей и выводам и стараться оценивать их на возможные искажения.
## Результат
### Создан готовый к машинному обучению датасет с более чем 15 значимыми признаками; выявлены корреляции между продуктовыми характеристиками и продолжительностью активности компании на рынке.
## Стек
### Python (pandas, NumPy, matplotlib, seaborn)

# Визуализация метрик сервиса доставки еды
[Дэшборд проекта](https://datalens.yandex/03afgd2ppyuil)
## Цель проекта 
### Цель проекта — проанализировать ключевые метрики сервиса доставки еды «Всё.из.кафе» в Саранске за период с 1 мая по 30 июня 2021 года.
## Описание данных
### «Всё.из.кафе» - сервис доставки еды в Саранске. Сервис позволяет пользователям просматривать меню ресторанов-партнёров, выбирать блюда и оформлять заказы с доставкой.
## Результат
### Выявлены устойчивые паттерны, определены топ-рестораны по LTV и состав популярных блюд и сформулированы рекомендации по улучшению сервиса.
## Стек
### SQL, DataLens
